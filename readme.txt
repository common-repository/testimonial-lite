== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use Testimonial in your post, page, widget or anywhere you like. How to create Testimonial? lets see.


1. Upload the folder testimonial-lite to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
3. Add testimonial going to wp-admin -> Testimonials Lite -> Add New Testimonial.
4. Call shortcode to your WordPress editor directly, by using
`
[testimonial-lite]
`
Call shortcode to your php template file using
`
<?php  echo do_shortcode('[testimonial-lite]'); ?>
`

Now save your page, then visit the newly created page & see something awesome. :)

Shortcode attributes
`
items			= use any number
items_desktop 		= use any number
items_tablet 		= use any number
items_mobile 		= use any number
total_items 		= use any number
order_by		= date, rand, title
order			= DESC / ASC
nav			= true / false
`
